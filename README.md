## 📜 游戏简介

**自由意志雷** 是一款融合了**命运哲学**与**随机博弈**的终端踩雷游戏。  
你将在天道的逼仄区间内一次次诵念数字，避开那道由你“亲手创造”却又无法预知的劫雷。

- 没有固定的雷位——**雷因你而生**。
- 没有注定的结局——**你的每一次输入，都在重塑天机**。
- 通关与否，看似由你抉择，却总透着一丝**不可名状的宿命感**。

---

## 🎯 核心玩法

1. **初始诵念**：输入任意字符（你的“天命之念”），游戏据此生成第一道雷的位置（0~100之间）。
2. **正式历劫**：
   - 输入一个 `0~100` 的整数。
   - 若数字等于当前雷位 → **应劫而亡**，游戏结束。
   - 若数字不等于雷位 → **渡劫成功**，天域收缩至你与雷位之间的某一侧（类似二分区间）。
3. **特殊指令**：
   - 输入 `0` → 切换界面色彩（阴阳二气流转）。
   - 输入 `100` → 窥探当前雷位的具体位置（仅本轮有效）。
4. **通关条件**：当天域区间长度 ≤ 2 时，再无雷位容身之处 → **飞升上界**！

## 🔮 自由意志 · 哲学内核

### ❓ 自由意志究竟在哪？

**你的每一次输入，就是你的自由意志** 

- 你诵出的数字决定了天域如何收缩，从而影响后续的雷位范围。
- 你的所有输入（包括初始的“天命之念”）会通过混沌算法共同决定**下一道雷**的具体位置。
- 你甚至可以主动窥探天机（输入 `100`），或切换阴阳（输入 `0`）——**一切选择都由你发出**。

### ❗ 那为什么又说“不由你决定”？

- 你无法精确知道下一道雷会落在区间的哪个坐标——它由你累积的自由意志与**微秒级时间、内存地址**等不可控却又已由固定因果链确定的因素纠缠而成。
- 当你自信满满地输入一个数字时，可能会发现它正好撞上那道“你自己生出的雷”。
- **你创造了命运，却无法完全掌控它**——这正是天道的微妙之处。

> 🌌 **哲思一刻**：  
> 自由意志并非“为所欲为”，而是**在不可预测的混沌中，你的每一次选择都真实地改变着世界的走向**。  
> 通关看似是你的策略所致，但换个初始的“念”，一切又将截然不同。  
> **宿命与自由，在此刻共舞。**

## 🖥️ 运行方式

### 环境要求
- Python 3.6 及以上
- 终端支持 ANSI 颜色（Windows 建议使用 Windows Terminal 或 Git Bash）

### 快速开始
```bash
# 下载或保存代码文件为 freedom_thunder.py
python freedom_thunder.py
```

### 操作说明
- 按照提示输入整数（或 `0` / `100`）。
- 游戏过程中可随时按下 `Ctrl + C` 主动弃劫（半途而废，道心不稳）。
- 每轮结束后可查看**天命回放**，回顾你与天道的每一次博弈。
